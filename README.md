## 遊戲玩法

| 動作 | 效果 |
|------|------|
| **深蹲**（完整一下）| 雷射瞄準最近的怪物 |
| **伏地挺身**（完整一下）| 雷射齊射最近 3 隻怪物 |

### 動作判定標準
- **深蹲**：膝蓋角度（髖-膝-踝）從 >160° 下降至 <100°，再回到 >160°
- **伏地挺身**：偵測到水平體位後，肘部角度（肩-肘-腕）從 >155° 下降至 <90°，再回升

### 遊戲機制
- 怪物從螢幕四邊不斷生成，向玩家逼近
- 怪物碰到你 → 扣一條命（共 3 條）
- 分數達到 100 的倍數 → 升等，怪物變更快更多
- 伏地挺身模式為 **三連射**（更強力）

## 技術細節

### 使用技術
- **OpenCV** — 影像擷取、繪製 HUD、特效渲染
- **MediaPipe Pose** — 33 點全身骨架偵測
- **NumPy** — 向量運算、角度計算

### 姿勢偵測原理
```
深蹲：
  髖 ─── 膝 ─── 踝
       ↑ angle
  站立 ≈ 170°, 蹲下 ≈ 80°

伏地挺身：
  肩 ─── 肘 ─── 腕
       ↑ angle
  撐起 ≈ 165°, 壓下 ≈ 80°
```

### 怪物種類
- 🧟 **殭屍**（綠色）- 標準速度
- 💀 **骷髏**（白灰）- 中速
- 🦇 **蝙蝠**（紫色）- 飛行浮動
